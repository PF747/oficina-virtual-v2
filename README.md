# 🦁 πφe Oficina Virtual V2

**Cyberpunk Space Command Center** — A futuristic 3D virtual office for the πφe AI organization.

## Features

### 🏢 3D Virtual Office
- Three.js powered space station environment
- 10 humanoid AI agents at their workstations
- 4 VIP areas: Director's Bridge, Trading Floor, War Room, Innovation Lab
- Post-processing bloom effects, nebula clouds, star field
- Interactive: click agents to fly camera, tooltips, auto-rotate

### 📈 Trading Command Center
- Phantom_Defi style dark neon dashboard
- Live candlestick charts with volume
- Simulated positions, order book, risk metrics
- 20 strategy cards with status indicators
- Real-time data simulation

### 🧠 20 Trading Strategies
Complete professional strategy library with:
- Entry/exit conditions
- Risk management rules
- Example setups with calculations
- Python implementation code

Strategies range from Momentum Breakout to Machine Learning Ensemble Signals.

### 🔒 Secure Server
- Express.js with session authentication
- bcrypt password hashing
- Rate limiting & helmet security headers
- VIP access roles

## Tech Stack
- **Frontend:** Three.js, Canvas API, vanilla JS
- **Backend:** Node.js, Express.js
- **Auth:** express-session, bcrypt
- **3D:** Three.js 0.170.0 with post-processing

## Run
```bash
npm install
node server.js
# → http://localhost:9443
```
